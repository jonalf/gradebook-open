    Bug Report/Feature Request

    Getting Started video

    • To log into gradebook, go to https://nibbler.stuy.edu/gradebook
    • Your browser may display a message stating that the page is untrusted or something along those lines. You may proceed anyway. The reason behind the message is that Gradebook uses what is called a self-signed certificate. This allows all information passed between Gradebook and your browser to be encrypted without having to pay an external company to validate the certificate on my end. If you have any concerns you may contact me.
    • You will be prompted to enter your username and password. If you do so incorrectly simply try again.

    Selction Page video

    • The top of the selection page will display the current term that you are working in. This will default to the current calendar term but will change if you select a class from a previous term.
    • The first drop-down menu will have a list of all the classes for the current term. Find the class and click Select Current Class to go to that class.
    • You can use the second drop-down menu to select a class from a previous term. Find the class you want and click Select Old Class.

    Classview Pages

    • After selecting a class, you will be taken to a page where you should see all your students in a grid. This is known as a Classview page. There are 6 different Classview modes. The active mode is always displayed at the top in the navigation bar.
    • Below the navigation bar, you will always see the current class and selected term. There will also be a ? button which will display a small help popup for the current mode.

      • Attendance Mode video

        • This is the default Classview mode. Whenever you select a class from the select page or navbar, this will be the active mode.
        • Here you will be able to view and edit the class attendance for a single day.
        • Clicking on a student will cycle through a series of colors. Each color represents a different attendance mark:
          • Red: Absent
          • Orange: Late
          • Green: Excused Absence
          • Blue: Excused Lateness
        • Any student without a colored border will be considered present. Gradebook only saves absence/lateness data.
        • Attendance marks will not be saved until you hit the Save Attendance button.
        • You can save attendance multiple times. If you change a student from absent (excused or not) to late (excused or not), the absence mark will be removed. If you change a student from a non-excused mark to an excused one, the non-excused mark will be removed.
        • You cannot change a student from absent/late (excused or not) to present from this mode. You may do so through the Student Mode.
        • Clicking Pick Date will bring up a small calendar. You can select any other date to view/edit the class attendance for that day.

      • Arrange Mode video

        • This mode will allow you to reseat your class as well as add/remove rows or columns.
        • The class must always be displayed as a grid, but you may use empty seats to create a non-grid based seating chart.
        • Clicking on a student (or empty space) will highlight them in red. Clicking on a second student (or empty space) will swap positions between the two. Every seat swap is immediately saved.
        • The Resize button will allow you to add/remove rows or columns to the seating chart. If you remove a row/column that has students in it, you will not be able to see them, but they will not have been removed. If you add back the row/column the students will appear again.

      • Student Mode video

        • This mode will allow you to view/edit data about a particular student. You will have access to all attendance data, all assignment data as well as some personal information.
        • The Add Student button will allow you to manually add a student to your class. You must include the student's first and last name and their OSIS. It is very important you enter their OSIS correctly, this will be their identification number for Gradebook.
        • Clicking on a student will bring up a window with more options.
        • The Student Information button will show you the student's name, nickname (if any), OSIS, 4-digit Stuy ID, homeroom and email address (if one is on file).
          • Clicking on Edit will allow you to change any of these, excpet for the student's name and OSIS.
          • Nicknames will be displayed above the student's name in all Classview pages
        • The Student Attendance button will display 4 caldenars. One each for absences, excused absences, latenesses and excused latenesses. You will aslo see the total number of each type next to the corresponding heading.
          • Each date shaded in dark gray represents a mark of that type in the student's record. You can toggle individual days on or off by clicking on them.
          • Clicking Change Attendance will save any changes you have made. This is the only way to mark a student present from any other mark by unselecting a day from any of the calendars.
        • The Student Work button will show you an overview of the student's grades for each assignment type you have saved (assignment types can be changed in the Grades mode).
          • Clicking on Full View will allow you to see a detailed list of all the assignments for that type. You can see any missing or excused assignments, as well as each completed assignment and its grade.
          • Clicking on an individual assignment will allow you to change its grade.
        • Clicking Remove Student will remove the student from the current class. This is an irreversible action. Removing a student deletes all the information pertaining to that student.
        • Clicking Transfer Student will allow you to transfer the student to another one of your current classes. If the seating layout for the new class does not have room, you will be asked to resize the layout for that class before you can transfer the student.

      • Assignment Mode video

        • This mode will allow you to enter and modify grades for any assignment.
        • Each student will have a box under their name where you can enter a grade. The grade must be a number or the letter e (case does not matter). If you enter e, then the student will be excused for that assignment and it will not factor into their total grade.
        • On the right had side you will be able to enter an assignment name, the maximum points for the assignment and the default value (this will be applied if you don't enter a grade for a particular student). Make sure that the assignment name has not already been used for that class.
        • You must also select what type of assignment you are entering by selecting it from the options below the default grade.
        • Clicking Save Grades will save the grades and take you to the Grades mode Classview page.
        • If an assignment has already been saved, clicking Delete Assignment will remove the current assignment from all records. This is an irreversible action.
        • You will also see a dropdown menu that contains all saved assignments. Selecting an assignment from this menu and clicking Change Assignment will bring up all the information for that assignment. You can then change individual grades or the max points and save these changes (or delete the entire assignment).

      • Grades Mode video

        • This mode will display every student's current grade. You will also be able to add/remove assignment types, change the weights for each type and setup an option to drop the lowest test grade.
        • If at any time the grades do not appear under each student, you can click Change Weights, this will refresh the view. Grades not displaying sometimes is a known bug and is being looked at.
        • Any student getting a failing grade (below 65) will be highlighted in red. Any student with a grade between 75 and 65 will be highlighted in orange.
        • On the right hand side of the window you will see each assignment type and the current weight assigned to that type. You can enter any weights you'd like, but make sure that the total of all weights is 1.0 (100%). Clicking Change Weights will change the weights to the entered values.
        • The More Options button will allow you to drop the lowest test grade either by percentage or total points for all students. You can toggle this on or off. This will only effect grades in the tests category.
        • Add Assignment Type will allow you to create a new assignment type. You will be promted for a name, just make sure it is not a name used for an existing assignment type.
        • Remove Assignment Type will bring up a list of the current assignment types and allow you to remove one. This will remove all data for that assignment from the system, including student grades. This action is irreversible.

      • Participation Mode video

        • This mode currently has no bearing on a student's grade. Nor does any of the information get saved. It is a work in progress.
        • Random Student will highlight a randomly selected student. It will cycle through all the students in a class before it picks another one.
        • Each student has a + and - button next to their name. Clik the + button to recognize good participation and the - number to recognize poor participation. Clicking Weighted will select a random student, but those with lower participation totals are more likely to be selected.

    Reports Pages video

    • Reports pages will display your students in an alphabetically sorted list in order to display certain information.
    • There are 3 different Reports pages, the active page is always displayed in the top navigation bar.

      • Today's Attendance

        • A table will be displayed for each of your current classes. Each student will either have Absent, Late or nothing next to their name.
        • You can view the attendance totals by entering a different date in the text box above the first class table and clicking on the View Day button. You can either type in the date (format: mm-dd-yyyy), or select it from a calendar that will appear once you click in the text box.

      • Attendance Totals

        • A table will be displayed for the current class only. It will contain all the students in that class and the total number of absences, latenesses, excused absences and excused latenesses for each.

      • Grades

        • This page will only display the grades of the current class.
        • The first table on this page shows the total percentage for each assignment type, the raw grade (calculated using the wieghts entered in the Grades Classview page) and then the grade rounded to the nearest multiple of 5 for grades less than 90 or the nearest integer for grades above 90.
        • Below the overall grade table will be a table for each assignment type that exists for the current class. Each table will display all the assignment grades for every student.
        • All the tables on this page will have the average and median values for each column of data below the last student's information.

    Backups video

    • Each option on this page can be done for the current class only or for all classes in the selected term.

    • Downloading a Backup File

      • You can either download a backup csv (comma separated value) file or a raw data backup file.
      • A csv file can be opened in any spreadsheet application. The information will be in a similar format to the tables in the reports view.
      • The raw data file should only be used in conjunction with the restore feature on this page.

    • Restoring from a Backup

      • In the event that something happens to your class data, you can restore it from a previouly saved backup. To do so, select the appropriate backup file and click restore.
      • You can only restore from a raw data backup file, not a csv backup file.

    Changing Your Password video

    • In order to change your password, select Change Password from the navigation bar. You must enter your current password once and your new password twice.
    • All passwords must be at least 6 characters long.

    Allowing Students To See Their Grades

    • Students can look at their grades by going to: nibbler.stuy.edu/gradebook/studentlogin
    • If it is the first time logging in, they must enter their OSIS and click "login". They will then be sent to a page where they can set a password. Once set, they will be brought back to the login page.
    • Once the student has created a password, they can log in and see any grades that have been entered in any class where that teacher is using Gradebook. Students do not need separate accounts for different classes.
    • Assignments are not automatically shared
